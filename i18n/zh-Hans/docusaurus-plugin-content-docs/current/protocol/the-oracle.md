# 预言机

协议的某些部分需要不存在于比特币上的信息，其中最重要的就是**价格**。举例来说，FIRE 债券定价就需要一个参考价格才能运作。SUBFROST 通过一个去中心化的**预言机**来提供这类信息。

## 它是如何运作的

预言机并不信任单一服务器上报的数值，而是以一个小型**分布式群体**的形式运行。每个参与者各自独立地读取相关的外部数据源和当前的链上状态，群体必须先达成一致，一个数值才能被提交上链。任何单一参与者都无法单方面决定结果。

由于该数值是由群体中达到门限数量的参与者共同商定并发布上链的，依赖它的合约（例如 [FIRE](../using-subfrost/fire-vault) 债券机制）就能够读取到一个能够抵御任何一方操纵的价格。

## 应用场景

- **FIRE 债券。** 债券价格锚定于预言机上报的参考价格，并设有与协议资金库挂钩的价格下限，因此债券不会以低于其背书价值的价格被出售。
