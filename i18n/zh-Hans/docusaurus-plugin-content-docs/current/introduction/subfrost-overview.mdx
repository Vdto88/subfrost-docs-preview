# SUBFROST 概述
比特币 DeFi 是金融领域最具前景的前沿方向之一。然而，原生 BTC 并不能直接兼容 Bitcoin L1 生态系统中的高级应用，包括 AMM、借贷协议，或是 zk 混币器等隐私解决方案。

SUBFROST 通过发行合成资产来释放这一潜力，使原生 BTC 能够无缝驱动下一代 DeFi。


## 什么是 SUBFROST？
SUBFROST 是一个去中心化托管方，能够在 Bitcoin L1 上实现无需信任的 DeFi 生态系统。

它由 [**Alkanes**](/protocol/alkanes) 智能合约协议驱动，直接在 Bitcoin 上发行合成资产，因此你的 BTC 无需离开 Bitcoin 就能与链上应用交互。

如果没有它，"比特币 DeFi"将继续依赖需要信任的跨链替代方案，而我们完全不会将其视为"比特币 DeFi"。
