# API Platform Overview

The Subfrost API Platform provides a comprehensive dashboard for managing your API access, monitoring usage, and configuring authentication.

## Dashboard Features

### API Keys Management

Create and manage multiple API keys for different applications:

- Generate new keys with custom names
- View key usage statistics
- Revoke compromised keys
- Set per-key rate limits (Business)

### Domain Management

Configure CORS-based authentication:

- Add and verify domains
- Automatic origin verification
- Support for subdomains and wildcards
- Domain-specific usage tracking

### Usage Analytics

Monitor your API consumption:

- Real-time request counts
- Historical usage graphs
- Method-level breakdown
- Error rate tracking

### Billing & Plans

Manage your subscription:

- View current plan details
- Upgrade or downgrade
- Payment history
- Usage-based billing (Business)

## Account Types

### Free Tier

- 1,000 requests per day
- 60 requests per minute
- 1 API key
- Community support

### Pro Plan ($45-$55/mo)

- 50,000 requests per day
- 600 requests per minute
- 10 API keys
- Email support
- Priority routing

### Business ($90-$120/mo)

- Unlimited requests
- Unlimited API keys
- Custom rate limits
- Dedicated support
- SLA guarantees
- Custom endpoints

## Security Features

- **Key Rotation**: rotate keys without downtime
- **Domain Verification**: verify domain ownership
- **Usage Alerts**: get notified of unusual activity
- **Audit Logs**: track all API key operations
